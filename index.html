🤖 Models Implemented

7 models trained — both on original imbalanced and SMOTE-balanced data

📐
Ordinal Logistic Regression
Custom scipy-based implementation

Proportional-Odds / Cumulative Link model built from scratch. Models cumulative probabilities P(Y ≤ k) for each threshold. Optimized via L-BFGS-B minimization.

🌲
Random Forest
500-tree ensemble classifier

Ensemble of 500 decision trees with max depth 10. Class-weighted to compensate for imbalance. Also retrained on SMOTE-balanced data.

🚀
Gradient Boosting (GBM)
300 sequential estimators

Sequential ensemble with learning rate 0.1 and stochastic subsampling (80%). Typically outperforms RF on tabular data. Trained on SMOTE data.

🧠
Multi-Layer Perceptron
Deep neural network (128→64→32)

3-hidden-layer neural network with ReLU activation, early stopping, and L2 regularization. Demonstrates deep learning limitations on small datasets.

⚡
Support Vector Machine
RBF kernel with probability calibration

SVM with radial basis function kernel and balanced class weights. Probability calibration enables reliable confidence estimates.

📊
Logistic Regression
sklearn baseline for CV comparison

Standard multinomial logistic regression used as a baseline reference for cross-validation comparison against the Random Forest ensemble.

📋 Methodology & Analysis

📈
Statistical Analysis
  • ✅ Chi-Square Tests of Independence (7 variables)
  • ✅ Bayes' Theorem Conditional Probabilities (5 conditions)
  • ✅ Odds Ratios from OLR Coefficients
  • ✅ Wilson 95% Confidence Intervals
  • ✅ ROC Curves with AUC (per severity class)
  • ✅ Brier Score Calibration
  • ✅ Permutation Feature Importance
⚙️
Feature Engineering
  • 🔧 Speed Ratio (precrash / limit) — speeding indicator
  • 🔧 Speed × Night interaction term
  • 🔧 Incident Year & Month temporal features
  • 🔧 Vehicle Age indicator (Is_OldVehicle)
  • 🔧 6 binary condition flags (night, wet, highway, dark, bad weather, airbag)
  • 🔧 One-hot encoding of categorical variables
  • 🔧 SMOTE oversampling for class balance
