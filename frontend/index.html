🤖 Models Implemented

7 models trained — both on original imbalanced and SMOTE-balanced data

📐
Ordinal Logistic Regression
Custom scipy-based implementation

Proportional-Odds / Cumulative Link model built from scratch. Models cumulative probabilities P(Y ≤ k) for each threshold.

🌲
Random Forest
500-tree ensemble classifier

Ensemble of 500 decision trees with max depth 10. Class-weighted to compensate for imbalance.

🚀
Gradient Boosting
300 sequential estimators

Sequential ensemble with learning rate 0.1 and stochastic subsampling. Trained on SMOTE data.

🧠
Multi-Layer Perceptron
Neural network (128→64→32)

3-hidden-layer neural network with ReLU activation, early stopping, and L2 regularization.

⚡
Support Vector Machine
RBF kernel with probability calibration

SVM with radial basis function kernel and balanced class weights.

📊
Logistic Regression
sklearn baseline for CV

Standard multinomial logistic regression used as baseline for cross-validation comparison.
